# Avusturya Fotoğrafları

Bu klasöre Avusturya ile ilgili yüksek çözünürlüklü fotoğraflar ekleyin:

## Önerilen Fotoğraf Türleri:
- Viyana üniversite kampüsleri
- Avusturya üniversitelerinin binaları
- Öğrenci yaşamı fotoğrafları
- Viyana şehrinin genel görünümleri

## Dosya Adlandırma:
- `austria-hero.jpg` - Ana hero görseli
- `austria-campus.jpg` - Kampüs görseli
- `austria-student-life.jpg` - Öğrenci yaşamı
- `austria-education.jpg` - Eğitim görseli

## Teknik Özellikler:
- Çözünürlük: En az 1920x1080
- Format: JPG veya PNG
- Dosya boyutu: 2MB'den küçük
