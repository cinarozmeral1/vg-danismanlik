# Çek Cumhuriyeti Fotoğrafları

Bu klasöre Çek Cumhuriyeti ile ilgili yüksek çözünürlüklü fotoğraflar ekleyin:

## Önerilen Fotoğraf Türleri:
- Prag üniversite kampüsleri
- Çek Cumhuriyeti üniversitelerinin binaları
- Öğrenci yaşamı fotoğrafları
- Prag şehrinin genel görünümleri

## Dosya Adlandırma:
- `czech-hero.jpg` - Ana hero görseli
- `czech-campus.jpg` - Kampüs görseli
- `czech-student-life.jpg` - Öğrenci yaşamı
- `czech-education.jpg` - Eğitim görseli

## Teknik Özellikler:
- Çözünürlük: En az 1920x1080
- Format: JPG veya PNG
- Dosya boyutu: 2MB'den küçük
