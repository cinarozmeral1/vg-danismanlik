# Almanya Fotoğrafları

Bu klasöre Almanya ile ilgili yüksek çözünürlüklü fotoğraflar ekleyin:

## Önerilen Fotoğraf Türleri:
- Berlin, Münih, Hamburg gibi şehirlerin üniversite kampüsleri
- Almanya üniversitelerinin binaları
- Öğrenci yaşamı fotoğrafları
- Almanya şehirlerinin genel görünümleri

## Dosya Adlandırma:
- `germany-hero.jpg` - Ana hero görseli
- `germany-campus.jpg` - Kampüs görseli
- `germany-student-life.jpg` - Öğrenci yaşamı
- `germany-education.jpg` - Eğitim görseli

## Teknik Özellikler:
- Çözünürlük: En az 1920x1080
- Format: JPG veya PNG
- Dosya boyutu: 2MB'den küçük
