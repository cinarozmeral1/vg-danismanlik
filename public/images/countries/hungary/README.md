# Macaristan Fotoğrafları

Bu klasöre Macaristan ile ilgili yüksek çözünürlüklü fotoğraflar ekleyin:

## Önerilen Fotoğraf Türleri:
- Budapeşte üniversite kampüsleri
- Macaristan üniversitelerinin binaları
- Öğrenci yaşamı fotoğrafları
- Budapeşte şehrinin genel görünümleri

## Dosya Adlandırma:
- `hungary-hero.jpg` - Ana hero görseli
- `hungary-campus.jpg` - Kampüs görseli
- `hungary-student-life.jpg` - Öğrenci yaşamı
- `hungary-education.jpg` - Eğitim görseli

## Teknik Özellikler:
- Çözünürlük: En az 1920x1080
- Format: JPG veya PNG
- Dosya boyutu: 2MB'den küçük
