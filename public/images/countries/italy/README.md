# İtalya Fotoğrafları

Bu klasöre İtalya ile ilgili yüksek çözünürlüklü fotoğraflar ekleyin:

## Önerilen Fotoğraf Türleri:
- Roma, Milano, Floransa gibi şehirlerin üniversite kampüsleri
- İtalya üniversitelerinin tarihi binaları
- Öğrenci yaşamı fotoğrafları
- İtalya şehirlerinin genel görünümleri

## Dosya Adlandırma:
- `italy-hero.jpg` - Ana hero görseli
- `italy-campus.jpg` - Kampüs görseli
- `italy-student-life.jpg` - Öğrenci yaşamı
- `italy-education.jpg` - Eğitim görseli

## Teknik Özellikler:
- Çözünürlük: En az 1920x1080
- Format: JPG veya PNG
- Dosya boyutu: 2MB'den küçük
