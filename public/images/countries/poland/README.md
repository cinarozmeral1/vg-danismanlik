# Polonya Fotoğrafları

Bu klasöre Polonya ile ilgili yüksek çözünürlüklü fotoğraflar ekleyin:

## Önerilen Fotoğraf Türleri:
- Varşova, Krakow gibi şehirlerin üniversite kampüsleri
- Polonya üniversitelerinin binaları
- Öğrenci yaşamı fotoğrafları
- Polonya şehirlerinin genel görünümleri

## Dosya Adlandırma:
- `poland-hero.jpg` - Ana hero görseli
- `poland-campus.jpg` - Kampüs görseli
- `poland-student-life.jpg` - Öğrenci yaşamı
- `poland-education.jpg` - Eğitim görseli

## Teknik Özellikler:
- Çözünürlük: En az 1920x1080
- Format: JPG veya PNG
- Dosya boyutu: 2MB'den küçük
